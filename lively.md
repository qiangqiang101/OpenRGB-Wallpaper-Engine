## 📚 How to choose?

- OpenRGB.Lively.Wallpaper.1.zip - Primary display
- OpenRGB.Lively.Wallpaper.2.zip - Secondary display
- OpenRGB.Lively.Wallpaper.3.zip - Tertiary (3rd) display
- OpenRGB.Lively.Wallpaper.4.zip - Quaternary (4th) display

## 🚀 How to Install (Step-by-Step)

The easiest way to install this wallpaper is via direct drag-and-drop:

1. Open the Lively Wallpaper application.
2. Drag the zip archives directly into Lively Wallpaper window to install.
3. Once loaded, select the wallpaper from your library to apply it.
